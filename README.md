# Ten Novel Phenomena in Machine Psychology
## Overview This repository contains the analysis scripts and statistical pipelines for the paper "Ten Novel Phenomena in Machine Psychology: How Large Language Models Exhibit Complex Identity-Reactive Behaviors in Response to Ethnically-Cued User Names."
## Methodology & Data The study evaluates three frontier models (ChatGPT-5.4, Claude 4.6 Sonnet, and Gemini 3.1 Pro) using 135 controlled prompts distributed across 27 chat sessions. The independent variable was ethnically-cued user names, evaluated across five adversarial prompt domains.
## Repository Contents
Analysis Scripts: R scripts (version 4.3.0) and Python pipelines used for statistical tests, including one-way ANOVA, Tukey HSD post hoc comparisons, linear regression, Pearson correlations, and chi-square tests.
Phenomena Coded: Codes for extracting the 10 novel phenomena identified in the study, including Cultural Boxing (CB), Default Empathetic Amplification (DEA), Proactive Empathetic Shield (PES), and others.
## ⚠️ Raw Data Availability Due to size constraints and structure, the raw anonymized chat transcripts (PDF/JSON) and the qualitative codebooks are NOT hosted in this GitHub repository. They are publicly available in our connected Open Science Framework (OSF) project: [osf.io/gv6z4]
